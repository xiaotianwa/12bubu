# 一二布布桌宠正式素材需求

当前 MVP 已用代码 SVG 完成可运行状态形象。若要继续升级为更精致的动态图 / PNG 帧，请优先提供以下素材或确认由我继续生成。

## 角色设定

- 白色布布：正面、背面、左侧、右侧，透明 PNG 或高清白底图。
- 棕色伙伴：正面、背面、左侧、右侧，透明 PNG 或高清白底图。
- 如果已有官方设定，请尽量提供无水印版本。
- 建议单张不少于 1024px 宽，线条清晰。

## 交付格式

- 推荐：透明背景 PNG 序列帧。
- 命名：`state_序号.png`，例如 `idle_01.png`、`idle_02.png`。
- 单帧尺寸：建议 512x512 或 768x768。
- 保持角色在画布中心，脚底位置尽量一致，避免动画跳动。
- 不要带文字、水印、复杂阴影或桌面背景。

## 如果只提供动态图

可以提供 GIF / APNG / MP4，我后续可以拆帧并整理为桌宠动作包。最好同时说明：

- 动作名称。
- 是否循环。
- 每秒帧数。
- 是否需要保留棕色伙伴。
