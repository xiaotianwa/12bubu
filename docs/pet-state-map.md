## QQ 企鹅桌宠式行为

- 常驻形象使用官方同框待机素材，降低“临时绘制感”。
- 随机小动作会触发表情包动图，如白眼、转圈、骑车。
- 散步 / 骑车状态会轻微移动桌宠窗口，让它更像真的在桌面上活动。
- 单击布布触发跳舞动图反馈。
- 双击 / 右键打开功能轮盘。

## 交互联动

- 主桌宠双击 / 右键打开功能轮盘。
- 点击功能轮盘后，主桌宠会短暂切换到对应功能状态。
- 点击“投喂”“伙伴”不会打开面板，只播放陪伴型表情动作。
- 点击“表情”不会打开面板，会在动态表情参考动作里随机播放一个。
- 每个功能浮窗标题区也显示对应的小布布状态形象。
- 无操作一段时间后进入睡觉状态。
